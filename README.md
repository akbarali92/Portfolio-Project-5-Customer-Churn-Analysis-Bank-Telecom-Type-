The goal of this project is to explore customer data, calculate churn rate, and discover the key factors responsible for customers leaving the company.

* How can we reduce customer churn?

Some important findings from analysis:

* Certain age groups show higher churn risk.
* Balance alone does not strongly predict churn.
* Both male and female customers show churn behavior, but distribution varies.
* Churn rate is around 27%, which is significant for business.
